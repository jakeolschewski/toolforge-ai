## Best Practices

### 1. Start with Top Performers

**Priority Order:**

1. **Week 1:** Apply to top 5
   - Jasper
   - Copy.ai
   - Writesonic
   - Leonardo.ai
   - Notion

2. **Week 2:** Apply to next 10
   - Descript, Pictory, Grammarly, etc.

3. **Week 3:** Networks
   - Impact Radius
   - ShareASale

4. **Week 4:** Long tail
   - Individual tool programs

### 2. Disclosure Best Practices

**Required by FTC:**

On every page with affiliate links:

```html
<div class="bg-blue-50 border border-blue-200 p-4 rounded-lg mb-6">
  <p class="text-sm text-blue-800">
    <strong>Affiliate Disclosure:</strong> This page contains affiliate links.
    We may earn a commission when you purchase through these links at no extra
    cost to you. This helps us maintain this free resource.
  </p>
</div>
```

**On individual links:**
```html
<a href="..." rel="nofollow sponsored">
  Try Jasper (affiliate link) →
</a>
```

### 3. Link Management

**Use UTM Parameters:**

```
https://partner.jasper.ai/aff?ref=toolforge&utm_source=toolforge&utm_medium=affiliate&utm_campaign=jasper-review
```

**Track in Database:**
```sql
UPDATE tools
SET
  affiliate_link = 'https://partner.jasper.ai/aff?ref=toolforge',
  affiliate_program = 'Jasper Direct',
  commission_rate = '30%',
  commission_type = 'recurring'
WHERE slug = 'jasper-ai';
```

### 4. Create Comparison Content

**High-Converting Content Types:**

1. **Tool vs Tool:**
   - "Jasper vs Copy.ai: Which AI Writer is Better?"
   - Include affiliate links to both
   - Fair comparison

2. **Top 10 Lists:**
   - "10 Best AI Writing Tools for 2026"
   - Each with affiliate link
   - Honest rankings

3. **Use Case Guides:**
   - "Best AI Tools for Copywriters"
   - Recommend 3-5 tools
   - Affiliate links

### 5. Optimize for Conversions

**Elements that Increase CTR:**

1. **Strong CTAs:**
```html
<a href="..." class="btn btn-primary">
  Try Jasper Free for 7 Days →
</a>
```

2. **Trust Signals:**
```html
<div class="trust-badge">
  ✓ Free 7-day trial
  ✓ No credit card required
  ✓ Cancel anytime
</div>
```

3. **Urgency (If True):**
```html
<p class="text-red-600">
  Limited time: Save 20% on annual plans
</p>
```

4. **Social Proof:**
```html
<p class="text-gray-600">
  Join 100,000+ users who love Jasper
</p>
```

## Tracking & Optimization

### Key Metrics

**Track for Each Program:**

1. **Clicks:** Total affiliate link clicks
2. **CTR:** Clicks / Page Views
3. **Conversions:** Actual sales
4. **Conversion Rate:** Sales / Clicks
5. **Revenue:** Total earned
6. **EPC:** Earnings Per Click

**Example:**

| Program | Clicks | CTR | Conversions | Conv Rate | Revenue | EPC |
|---------|--------|-----|-------------|-----------|---------|-----|
| Jasper | 150 | 5.2% | 8 | 5.3% | $234 | $1.56 |
| Copy.ai | 85 | 3.1% | 3 | 3.5% | $56 | $0.66 |

### Optimization Strategy

**Monthly Review:**

1. **Identify Winners:**
   - EPC > $1.00 = Excellent
   - EPC $0.50-1.00 = Good
   - EPC < $0.50 = Needs improvement

2. **Double Down:**
   - Create more content for high EPC programs
   - Feature prominently
   - Add to category pages

3. **Improve Losers:**
   - Better CTAs
   - More detailed reviews
   - Comparison content
   - Better placement

4. **Cut Dead Weight:**
   - Remove links with 0 conversions after 3 months
   - Replace with better programs

### A/B Testing

**What to Test:**

1. **CTA Text:**
   - A: "Try Jasper Free"
   - B: "Start Writing with AI"
   - Winner: Measure CTR

2. **Button vs Text Link:**
   - A: Blue button
   - B: Inline text link
   - Winner: Measure clicks

3. **Placement:**
   - A: Top of review
   - B: After pros/cons
   - Winner: Measure conversions

**Tools:**
- Google Optimize (free)
- Vercel Edge Config
- Manual tracking
