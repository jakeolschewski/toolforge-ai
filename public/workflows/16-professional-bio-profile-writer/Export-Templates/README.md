# Professional Bio & Profile Writer - Export Templates

## Template 1: Multi-Platform Bio Package (Google Docs / Word)

**Format:** .docx or Google Docs template

**Description:** A single document containing all bio versions organized by platform, with clear section headers, word counts, and last-updated timestamps.

**Setup Instructions:**
1. Create a document with the following section headers: Micro Bio (50 words), Short Bio (150 words), Standard Bio (300 words), Extended Bio (500+ words)
2. Under each section, include fields for: Platform Target, Tone, Word Count, Last Updated, Approved By
3. Use a table of contents for easy navigation
4. Include a "Master Information" section at the top with the subject's verified credentials, achievements, and approved personal details
5. Save as a template (.dotx) for reuse across multiple subjects

## Template 2: LinkedIn-Optimized Profile Export (Plain Text)

**Format:** .txt with platform-specific formatting notes

**Description:** A plain text file formatted specifically for LinkedIn copy-paste, including the summary section, headline options, and experience section narrative snippets.

**Setup Instructions:**
1. Create a plain text file with sections: Headline (120 chars max), Summary (2,600 chars max), Experience Narratives (per role)
2. Include 3 headline variations for A/B testing
3. Add keyword density notes at the bottom (target keywords and their placement)
4. Format with line breaks that will render correctly in LinkedIn's text editor
5. Include a companion notes section with posting instructions and optimization tips
