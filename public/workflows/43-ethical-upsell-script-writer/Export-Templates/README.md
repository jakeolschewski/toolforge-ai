# Export Templates - Ethical Upsell Script Writer

## Overview
These templates help you organize, deploy, and track your ethical upsell scripts across different channels and formats.

---

## Template 1: Upsell Script Library Spreadsheet

**Purpose:** Centralized repository for all upsell scripts with metadata, status tracking, and performance notes.

**Columns:**
- Script ID
- Script Name
- Touchpoint (Checkout / Email / Support / Renewal / Other)
- Target Product (what the customer already has)
- Upsell Product (what is being offered)
- Price Point
- Framework Used (PAS / AIDA / Value-Stack / Other)
- Script Text (full copy)
- Ethics Audit Score (1-10)
- Deployment Date
- Status (Draft / Active / Paused / Retired)
- Conversion Rate
- Refund Rate
- Notes

**Setup Instructions:**
1. Create a spreadsheet with the columns listed above
2. Add one row per upsell script
3. Color-code status column (green = active, yellow = draft, red = retired)
4. Update performance metrics weekly
5. Add conditional formatting to flag refund rates above your acceptable threshold
6. Create a filtered view for each touchpoint type

---

## Template 2: Script Deployment Brief

**Purpose:** A one-page document for each script that provides deployment context, ethical clearance, and rollback instructions.

**Sections:**
- Script Name and ID
- Deployment Channel and Placement
- Target Audience Segment
- Ethical Clearance (date, reviewer, audit score)
- Go-Live Date and A/B Test Details
- Success Metrics and Thresholds
- Rollback Trigger Conditions
- Rollback Instructions
- Review Schedule

**Setup Instructions:**
1. Create a one-page document template with the sections above
2. Complete one brief per script before deployment
3. Require sign-off from at least one reviewer on the ethical clearance section
4. Store briefs alongside their corresponding scripts in your central repository
5. Reference the brief during performance reviews to compare expectations vs. results

---

## Template 3: Monthly Performance Report

**Purpose:** A recurring report that summarizes upsell performance, ethical compliance, and optimization opportunities.

**Sections:**
- Reporting Period
- Executive Summary (3-5 sentences)
- Script Performance Table (script name, conversion rate, refund rate, revenue impact, satisfaction score)
- Ethical Compliance Summary (any complaints, audit scores, guideline violations)
- Top Performer Spotlight (what is working and why)
- Underperformer Analysis (what is not working and proposed changes)
- Customer Feedback Highlights (positive and negative)
- Recommendations for Next Month
- Scripts to Retire, Update, or Create

**Setup Instructions:**
1. Create the report template in your preferred document format
2. Schedule a monthly data pull from your analytics platform
3. Populate the performance table with current metrics
4. Write the executive summary and analysis sections based on the data
5. Share with stakeholders and archive for trend analysis
6. Use the recommendations section as input for next month's script creation
