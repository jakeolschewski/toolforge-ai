# Export Templates - Email Response Optimizer

## Overview

These export templates help you integrate the outputs of the Email Response Optimizer workflow into your existing tools and systems. Choose the format that best matches your workflow.

---

## Template 1: Google Docs / Word Document

**Best for:** Detailed documentation, sharing with stakeholders, archival

**Setup Instructions:**
1. Create a new Google Doc (or Word document)
2. Set up the following sections:
   - Title and date
   - Executive summary / overview
   - Main findings or outputs
   - Action items with owners and deadlines
   - Next steps and follow-up schedule
   - Appendix (supporting details)
3. Apply consistent heading styles (H1, H2, H3)
4. Add page numbers and table of contents for longer documents
5. Share via link with appropriate permissions

**Tips:**
- Use the "Suggesting" mode for collaborative editing
- Enable version history to track changes
- Export as PDF for final distribution

---

## Template 2: Google Sheets / Excel Spreadsheet

**Best for:** Data tracking, scoring matrices, progress monitoring

**Setup Instructions:**
1. Create a new spreadsheet with these tabs:
   - **Dashboard** — Summary metrics and status at a glance
   - **Data** — Raw data and detailed entries
   - **Analysis** — Calculations, scoring, and rankings
3. Set up column headers based on your workflow outputs
4. Add data validation for consistent entries
5. Create charts or conditional formatting for visual insights

**Tips:**
- Use filters and pivot tables for quick analysis
- Protect formula cells to prevent accidental changes
- Set up automated calculations where possible

---

## Template 3: Notion / Project Management Tool

**Best for:** Team collaboration, ongoing tracking, integrated workflows

**Setup Instructions:**
1. Create a new Notion page (or project in your PM tool)
2. Set up as a database with relevant properties:
   - Status (Not Started / In Progress / Complete)
   - Priority (High / Medium / Low)
   - Owner (team member assignment)
   - Due Date
   - Tags/Categories
3. Create different views:
   - Board view for status tracking
   - Table view for detailed data
   - Calendar view for timeline items
4. Link to related pages and resources

**Tips:**
- Use templates for recurring entries
- Set up reminders for due dates
- Create a team dashboard page linking all workflow outputs

---

## General Export Guidelines

1. **Naming Convention:** Use format `[DATE]-email-response-optimizer-[VERSION]` (e.g., `2026-02-13-email-response-optimizer-v1`)
2. **Privacy:** Apply redaction guide before exporting to shared locations
3. **Versioning:** Always increment version numbers when making significant changes
4. **Access Control:** Set appropriate sharing permissions based on content sensitivity
5. **Backup:** Keep local copies of important exports
