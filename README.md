## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- Supabase account (free)
- Vercel account (free)

### 1. Clone & Install

```bash
git clone https://github.com/jakeolschewski/toolforge-ai.git
cd toolforge-ai
npm install
```

### 2. Set Up Supabase

1. Create project at [supabase.com](https://supabase.com)
2. Run migration: Copy `supabase-schema.sql` to SQL Editor
3. Get credentials from Settings â†’ API

### 3. Configure Environment

```bash
cp .env.example .env.local
# Edit .env.local with your Supabase credentials
```

### 4. Seed Database

```bash
npm run seed
```

Adds **91+ real AI tools** and **8+ quality reviews** in 1 minute!

### 5. Run Development Server

```bash
npm run dev
```

Visit `http://localhost:3000` - Your site is ready! ğŸ‰

### 6. Deploy to Vercel

```bash
vercel --prod
```

Or click: [![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/jakeolschewski/toolforge-ai)
